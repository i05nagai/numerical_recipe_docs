Householder Transformation¶

void recipe::linear_algebra::ComputeHouseholderVector(const double *vec_x, const int size_vec, double *householder_vec, double *householder_coeff)

Compute Householder Vector of vec_x. Calcualte $v$ and $\beta$ which satisfy.

$$ (I - \beta vv^{{T}})x = \|x\|_{2} e_{1} $$

where $(e_{i})$ is a standard basis of euclid space and $v = (1, v^{2}, \ldots, v^{2})$.

Parameters
  • vec_x: vector $x$.
  • size_vec: size of vector $x$ and $v$.
  • householder_vec: Householder vector $v$.
  • householder_coeff: Householder coefficient $\beta$.

std::unique_ptr<double[]> recipe::linear_algebra::ComputeHouseholderMatrix(const double *householder_vec, const double householder_coeff, const int size)

Compute Householder matrix.

$$ (I - \beta vv^{\mathrm{T}}) $$

where $\beta := 2 / \|v\|_{2}^{2}$ and $v$ is a householder vectror.

Return

Parameters
  • householder_vec: Householder vector $v$
  • householder_coeff: coefficient of Householder transformation $\beta$.
  • size: the size of vector $v$.

  • val2: value $b$
